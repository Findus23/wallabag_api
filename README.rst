============
Wallabag API
============

Python API for Wallabagap

Requirements :
==============
* Flask == 0.10.1
* requests == 2.5.0


Installation:
=============
to get the project, from your virtualenv, do :

.. code:: python

    git clone https://github.com/foxmask/wallabag-api/
